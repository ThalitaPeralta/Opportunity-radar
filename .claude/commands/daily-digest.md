# Daily Digest

Fetch all Jira tickets created or updated today for the product team.

For each ticket, run the following steps in order:

1. Rewrite the feedback as a problem statement using the framework in @context/problem-statement-guide.md
2. Assess the problem using the criteria in @context/assessment-criteria.md, considering @context/company-strategy.md and @context/product-cycle-strategy.md
3. Suggest follow-up discovery questions based on @references/discovery-questions.md
4. Draft a stakeholder reply following @context/stakeholder-reply-tone.md

Format the output for each ticket using the template in @output-templates/daily-digest-template.md

Then write all results to a new Notion page titled "Opportunity Radar — [today's date]".
