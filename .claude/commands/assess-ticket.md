# Assess Ticket

Given a problem statement, assess it using the criteria defined in @context/assessment-criteria.md.

Consider the current company strategy from @context/company-strategy.md and product cycle priorities from @context/product-cycle-strategy.md.

**Input:** paste the problem statement below.

**Output format:**
- **Business Impact**: [score + rationale]
- **User Experience Impact**: [score + rationale]
- **Strategy Alignment**: [score + rationale]
- **Overall Assessment**: [summary]
